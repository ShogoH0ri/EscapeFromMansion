## 概要
UnrealEngineで制作したFPSホラー探索ゲーム。プレイヤーは暗い屋敷を探索しながら鍵を集め、ゴールを目指します。

## 開発背景
学校の課題で、UnrealEngine作品を作成すること。この作品でクラス最優秀賞を取り、ゲームクリエイター甲子園2024での作品提出も行いました
